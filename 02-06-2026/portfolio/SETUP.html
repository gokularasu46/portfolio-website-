📋 Portfolio Setup Guide

Welcome to your personal portfolio! Follow these steps to get started quickly.

📁 Project Structure

Here's what your portfolio folder contains:

portfolio/
├── index.html ← Main website file
├── css/
│ └── styles.css ← Styling & layout
├── js/
│ └── script.js ← Interactivity & animations
├── assets/
│ ├── images/ ← Place your profile photo here
│ └── resume.pdf ← Your resume file
├── README.md ← Full documentation
├── QUICKSTART.md ← Quick setup guide
└── SETUP.html ← This file

Step 1: Prepare Your Assets

You'll need:

  • Profile Image - A professional photo (JPG or PNG)
    • Recommended size: 400x400px (square)
    • Save as: assets/images/profile.jpg
  • Resume PDF
    • Format: PDF
    • Save as: assets/resume.pdf

Step 2: Update Your Information

Edit index.html with your details:

  1. Open index.html in a text editor (VS Code, Notepad++, etc.)
  2. Find and replace:
    • Gokularasu K → Your name
    • AI & Data Analytics Enthusiast | Web Developer → Your designation
    • Hero section text with your introduction
    • About section with your details
    • Social media links (LinkedIn, GitHub, Twitter, Email)

Step 3: View Your Portfolio

Choose one method:

  1. Simple (Recommended for beginners)
    • Double-click index.html in your file explorer
    • Website opens in your default browser
  2. Python Server 
    cd portfolio
python -m http.server 8000
# Then visit: http://localhost:8000
  3. VS Code Live Server
    • Install "Live Server" extension
    • Right-click index.html
    • Select "Open with Live Server"

🎨 Customization

Change Colors

Edit css/styles.css (lines 4-19):

:root {
    --dark-blue: #0f1419;      /* Main background */
    --sky-blue: #00d4ff;       /* Primary accent */
    --text-light: #e8e8e8;     /* Text color */
}

Add More Sections

Skills, Projects, Education, and Contact sections are already in index.html. Update the placeholder text with your content.

🚀 Deploy Online

Make your portfolio live on the internet:

Option 1: GitHub Pages (Free)

  1. Create GitHub account at github.com
  2. Create new repository named yourusername.github.io
  3. Upload portfolio files to repository
  4. Your site is live at: https://yourusername.github.io

Option 2: Netlify (Free)

  1. Go to netlify.com
  2. Drag & drop your portfolio folder
  3. Get instant deployment link

Option 3: Vercel (Free)

  1. Go to vercel.com
  2. Import from GitHub
  3. Auto-deploy on every push

⚠️ Troubleshooting

  • Images not showing? Check file names and paths
  • Styles not applied? Clear browser cache (Ctrl+Shift+Delete)
  • Mobile menu not working? Check browser console (F12)
  • Links broken? Verify href attributes in HTML
