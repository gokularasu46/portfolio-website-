## About This Folder

This folder contains the profile image for your portfolio website.

## Setup Instructions

### 1. Prepare Your Image
- Select a professional headshot photo
- Make it **square** (equal width and height)
- Recommended size: **600x600 pixels** (minimum 400x400)
- Image format: JPG, PNG, or WEBP

### 2. Name the File
- Save as: **profile.jpg**
- Keep the filename exactly as shown

### 3. Place in This Folder
- Move the file to: `assets/images/profile.jpg`
- This exact location (don't rename the folder)

### 4. Refresh Your Portfolio
- Open or refresh `index.html` in your browser
- Your profile image should appear!

## Optimizing Image Size

### Using Online Tools
1. Visit [Pixlr](https://pixlr.com/), [Canva](https://canva.com/), or [Compressor.io](https://compressor.io/)
2. Upload your image
3. Crop to square
4. Resize to 600x600px
5. Compress for web
6. Download as JPG
7. Rename to `profile.jpg`
8. Place in this folder

### Using Command Line (Optional)
```bash
# Using ImageMagick (if installed)
convert original-image.jpg -resize 600x600 -gravity center -extent 600x600 profile.jpg
```

## Troubleshooting

**Image not showing?**
1. Check filename is exactly `profile.jpg`
2. Verify file is in `assets/images/` folder
3. Clear browser cache (Ctrl+Shift+Delete)
4. Hard refresh page (Ctrl+Shift+R)

**Image looks blurry?**
1. Use higher resolution image
2. Ensure image is actually square
3. Check file isn't corrupted

**File size too large?**
1. Compress image using Compressor.io
2. Reduce dimensions to 600x600
3. Use JPG format instead of PNG
